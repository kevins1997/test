<strong>Installation</strong>

-composer install
<br>
-npm install
<br>
-npm run dev
<br>
-copy .env.example and paste it in a new file called .env and place your database data
<br>
-php artisan key:generate
<br>
-php artisan migrate
<br>
-php artisan db:seed
<br>
-php artisan serve
